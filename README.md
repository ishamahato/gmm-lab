# 🎨 Graphics & Multimedia Lab Experiments

## 📖 Description
This repository contains implementations of various Computer Graphics algorithms and transformations as part of the Graphics and Multimedia Lab.

The aim of this project is to understand fundamental concepts like line drawing, transformations, clipping, and 3D operations through practical coding.

---

## 📌 Objectives
- Learn basic Computer Graphics concepts
- Implement line drawing algorithms
- Understand 2D & 3D transformations
- Apply clipping and windowing techniques

---

## 📂 Project Structure
- exp1.py → Basic drawing
- exp2.py → DDA Line Algorithm
- exp3.py → Bresenham Line Algorithm
- ...
- exp24.py → Advanced transformations

---

## ✨ Features
- Implementation of core graphics algorithms
- Supports 2D and 3D transformations
- Visualization using Python libraries
- Covers complete GMM lab syllabus

---

## 🛠️ Technologies Used
- Python 🐍
- OpenGL (via libraries)
- Matplotlib
